Internet-Einstellungen überprüfen

Es gibt ein Problem mit der Internet-Einwahlkonfiguration:

1. Öffnen Sie "Einwählen & Konfigurieren."

2. Stellen Sie sicher, dass ein ISP über "Neuen ISP hinzufügen" im ISP-Menü eingetragen ist.

3. Wenn kein Standard-ISP ausgewählt ist, wählen Sie den gewünschten ISP als Standard-ISP aus und klicken Sie auf das Kontrollkästchen "Als Standard-ISP festlegen".

4. Sobald das Problem behoben ist, können Sie im Menü "Ansicht" auf "Neu laden" klicken, um zu versuchen, diese Seite erneut zu laden.
